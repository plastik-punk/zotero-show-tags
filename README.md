# ZOTERO SHOW TAGS  
[![Using Zotero Plugin Template](https://img.shields.io/badge/Using-Zotero%20Plugin%20Template-blue?style=flat-square&logo=github)](https://github.com/windingwind/zotero-plugin-template)

Plugin for Zotero that shows assigned tags in overview

## How to use:

Download [latest release](https://github.com/plastik-punk/zotero-show-tags/releases/download/v1.0/zotero-show-tags.xpi)
In Zotero:   Tools -> Plugins -> Install Plugin From File...
             Right click on column headers and check "Tags"
